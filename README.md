# Terminais LMC

Landing page pública da marca **Terminais LMC**, criada para centralizar a divulgação dos aplicativos LMC e servir como link oficial da bio do Instagram **[@terminaislmc](https://www.instagram.com/terminaislmc/)**.

## Visão da marca

A **Terminais LMC** é uma marca de software voltada para aplicativos práticos, modernos e acessíveis, criados para ajudar pessoas a terem mais controle, organização e clareza no dia a dia.

**Frase central:**

> Ferramentas digitais para controle real.

**Posicionamento:**

> Apps para quem não vive no improviso.

## Aplicativos publicados

### Alpha Pro Terminal

Terminal para traders acompanharem operações, risco, estratégias e gestão com mais disciplina.

- Foco: trading, controle operacional, risco e organização
- Plataforma: Android / Play Store
- Link: https://play.google.com/store/apps/details?id=com.lmc.alphaproterminal

### Divide Aí

Aplicativo para organizar despesas pessoais e compartilhadas com mais clareza, controle e praticidade.

- Foco: finanças pessoais, despesas compartilhadas, grupos e organização
- Plataforma: Android / Play Store
- Link: **adicionar link oficial da Play Store**

## Objetivo deste projeto

Criar uma página simples, bonita e profissional para:

- Apresentar a marca Terminais LMC
- Explicar rapidamente cada aplicativo
- Direcionar usuários para a Play Store
- Reforçar confiança e identidade visual da marca
- Servir como link único da bio do Instagram

## Direção visual

A identidade visual deve seguir a linha criada para o Instagram:

- Fundo escuro/grafite
- Azul e ciano neon como destaque
- Cards modernos e responsivos
- Visual tecnológico, sério e confiável
- Botões grandes e claros para uso mobile
- Experiência simples, sem poluição visual

## Stack recomendada

Este projeto deve ser iniciado com:

- Vite
- React
- TypeScript
- CSS responsivo/mobile-first
- Deploy na Vercel

## Cuidados de comunicação

Para o **Alpha Pro Terminal**, a comunicação deve evitar promessas de lucro, renda garantida ou automação milagrosa.

O foco correto é:

- Controle
- Gestão de risco
- Organização
- Disciplina operacional
- Ferramenta de apoio ao trader

Frases recomendadas:

> O foco não é promessa. É controle.

> Ferramenta não substitui conhecimento, mas ajuda a operar com mais disciplina.
