# Terminais LMC

Landing page pública da marca **Terminais LMC**, criada para centralizar a divulgação dos aplicativos LMC e servir como link oficial da bio do Instagram **[@terminaislmc](https://www.instagram.com/terminaislmc/)**.

## Visão da marca

> Ferramentas digitais para controle real.

> Apps para quem não vive no improviso.

## Aplicativos publicados

### Alpha Pro Terminal

Controle, risco e disciplina para traders.

- Link: https://play.google.com/store/apps/details?id=com.lmc.alphaproterminal

### ControleAi

Despesas compartilhadas sem confusão.

- Link: https://play.google.com/store/apps/details?id=com.divideai.app

## Stack

- Vite
- React
- TypeScript
- CSS responsivo/mobile-first
- Deploy na Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```

O output fica na pasta `dist/`.

## Deploy na Vercel

1. Conecte o repositório na [Vercel](https://vercel.com)
2. Framework Preset: **Vite**
3. Build Command: `npm run build`
4. Output Directory: `dist`

## Placeholders

- `/public/og-image.png` – Imagem para compartilhamento em redes sociais (recomendado: 1200x630px)
- `/public/favicon.png` – Favicon do site
