Política de Privacidade

A Terminais LMC respeita sua privacidade. Esta política descreve como os dados dos usuários podem ser tratados nos aplicativos Alpha Pro Terminal e ControleAi.

1. Quais dados podem ser coletados

Os aplicativos podem coletar dados necessários para funcionamento, autenticação, assinatura, preferências e recursos do aplicativo. Isso pode incluir:

  • Dados de conta e autenticação (e-mail, identificadores)
  • Preferências e configurações do usuário
  • Dados de uso para funcionamento dos recursos do app
  • Informações relacionadas à assinatura e plano

2. Uso dos dados

Os dados coletados são utilizados exclusivamente para o funcionamento dos aplicativos, manutenção dos serviços, melhoria da experiência do usuário e suporte ao cliente quando solicitado.

3. Serviços de terceiros

Os aplicativos podem utilizar serviços de terceiros para funcionamento, como Google Play (distribuição e assinaturas), Firebase (autenticação, banco de dados, analytics), Supabase (banco de dados, autenticação) ou outros serviços quando aplicável. Cada serviço possui suas próprias políticas de privacidade e termos.

4. Não venda de dados pessoais

A Terminais LMC não vende, aluga ou compartilha dados pessoais dos usuários com terceiros para fins comerciais ou publicitários.

5. Segurança

Utilizamos práticas de segurança para proteger os dados dos usuários. No entanto, nenhum sistema de transmissão ou armazenamento de dados é completamente seguro. Fazemos esforços razoáveis para manter a segurança, mas não podemos garantir proteção absoluta.

6. Direitos do usuário

O usuário pode solicitar informações sobre seus dados, correção ou exclusão entrando em contato pelo e-mail abaixo. Atenderemos dentro do possível e conforme a legislação aplicável.

7. Alterações na política

Esta política pode ser atualizada periodicamente. Recomendamos consulta regular para estar ciente de eventuais alterações.

8. Contato

Para dúvidas, solicitações ou questões sobre privacidade, entre em contato pelo e-mail: alphaterminallmc@hotmail.com
